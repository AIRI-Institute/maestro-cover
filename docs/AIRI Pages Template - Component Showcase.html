Layout & Spacing Guide

Описание: Важные правила и рекомендации по структуре страницы, отступам между секциями, адаптивности и кастомизации цветов. Эта секция поможет вам правильно организовать контент на странице.

Spacing Between Sections (Отступы)

Bootstrap использует утилиты spacing для управления отступами. Стандартная схема для секций в этом шаблоне:

Класс Значение Когда использовать
mb-5 Margin-bottom: 3rem Для основных секций (самое частое)
py-5 Padding Y: 3rem Для секций с фоном или границами
mb-4 Margin-bottom: 1.5rem Между подсекциями внутри section
mb-3 Margin-bottom: 1rem Между заголовками и контентом
mt-4 Margin-top: 1.5rem Для code examples после контента 
<!-- Стандартная секция с отступом снизу -->
<section class="mb-5" id="my-section">
    <div class="container">
        <div class="row justify-content-center">
            <div class="col-lg-8">
                <h2 class="mb-4">Section Title</h2>

                <!-- Подсекция -->
                <div class="mb-4">
                    <h3 class="mb-3">Subsection</h3>
                    <p>Content here...</p>
                </div>
            </div>
        </div>
    </div>
</section>

<!-- Секция с вертикальным padding (для фонов) -->
<section class="py-5 mb-5" id="highlighted-section">
    <div class="container">
        ...
    </div>
</section>

Layout Structure (Структура разметки)

Стандартная структура страницы использует Bootstrap Grid System с центрированным контентом:

  • <section> — основной контейнер секции
  • <div class="container"> — ограничивает максимальную ширину
  • <div class="row justify-content-center"> — центрирует колонки
  • <div class="col-lg-8"> — основная колонка контента (66% ширины на больших экранах)
Важно: Используйте col-lg-8 для основного текстового контента. Для wide content (графики, таблицы) можно использовать col-lg-10 или col-12. 
<section class="mb-5">
    <div class="container">
        <div class="row justify-content-center">
            <div class="col-lg-8">
                <!-- Ваш контент здесь -->
            </div>
        </div>
    </div>
</section>

Responsive Design (Адаптивность)

Bootstrap breakpoints для адаптивного дизайна:

Breakpoint Class prefix Screen width
Extra small (none) <576px
Small sm ≥576px
Medium md ≥768px
Large lg ≥992px
Extra large xl ≥1200px

Примеры адаптивных классов:

  • col-12 col-md-6 col-lg-4 — полная ширина на мобильных, 50% на планшетах, 33% на десктопах
  • d-none d-lg-block — скрыть на мобильных, показать на десктопах
  • table-responsive — добавляет горизонтальный скролл для таблиц на маленьких экранах
<!-- Адаптивная сетка -->
<div class="row g-4">
    <div class="col-12 col-md-6 col-lg-4">
        <div class="card">...</div>
    </div>
    <div class="col-12 col-md-6 col-lg-4">
        <div class="card">...</div>
    </div>
</div>

<!-- Адаптивная таблица -->
<div class="table-responsive">
    <table class="table">...</table>
</div>

<!-- Адаптивная видимость -->
<div class="d-none d-lg-block">
    Видно только на больших экранах
</div>

Для изменения цветов шаблона отредактируйте файл public/styles/template.css и измените CSS-переменные:

Переменная По умолчанию Где используется
--bs-primary #2ebead Кнопки, ссылки, акценты
--bs-secondary #8c9090 Вторичные элементы
--bs-success #53d98a Успешные состояния, badges
--bs-info #64bbf3 Информационные блоки
--bs-body-font-family 'Inter', sans-serif Основной шрифт 
/* В файле public/styles/template.css */
:root {
    /* Измените основной цвет */
    --bs-primary: #your-color;
    --bs-primary-rgb: 255, 100, 50; /* RGB версия */

    /* Измените шрифт */
    --bs-body-font-family: 'Roboto', sans-serif;

    /* Измените размеры шрифтов */
    --bs-h1-font-size: 42px;
    --bs-body-font-size: 18px;
}

<div style="background: var(--bs-primary);">

/* Градиенты с CSS переменными */

Navigation with Scrollspy

Боковое навигационное меню с автоматическим выделением активного раздела при прокрутке страницы. Меню уже встроено в шаблон и автоматически появляется/скрывается.

💡 На этой странице используется такое меню! Посмотрите на левую часть экрана — там находится навигация по всем секциям примеров. При прокрутке активный пункт подсвечивается автоматически.
📝 Как добавить свои пункты в меню:
  1. Откройте файл index.html и найдите <nav id="page-navbar">
  2. Добавьте новый пункт в меню: <a class="nav-link" href="#my-section">Моя секция</a>
  3. Создайте секцию с соответствующим id: <section id="my-section">

Вложенные пункты: Используйте вложенный <nav> с классом ms-3 для отступа.

🗑️ Как скрыть меню: Если вы не хотите использовать навигационное меню, просто удалите элемент <nav id="page-navbar">...</nav> из вашего файла index.html.

Large language models (LLMs) are notorious for hallucinating, i.e., producing erroneous claims in their output. Python Example
import torch
import torch.nn as nn

class TransformerModel(nn.Module):
    def __init__(self, vocab_size, d_model=512, nhead=8):
        super().__init__()
        self.embedding = nn.Embedding(vocab_size, d_model)
        self.transformer = nn.TransformerEncoder(
            nn.TransformerEncoderLayer(d_model, nhead),
            num_layers=6
        )

    def forward(self, x):
        x = self.embedding(x)
        return self.transformer(x)

# Initialize model
model = TransformerModel(vocab_size=10000)
print(f"Model parameters: {sum(p.numel() for p in model.parameters())}")

            accuracy: data.metrics.accuracy,
            f1Score: data.metrics.f1_score,
            timestamp: new Date(data.timestamp)
        };
    } catch (error) {
        console.error('Failed to load results:', error);
        return null;
    }
}
